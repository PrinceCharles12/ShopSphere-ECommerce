# ShopSphere - E-Commerce Website

## Tech Stack
- ASP.NET Core Web API
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Pages
- Bootstrap 5
- AutoMapper
- Razorpay Payment Gateway

## Features

### Customer
- User Registration & Login
- Product Catalog
- Categories
- Wishlist
- Shopping Cart
- Checkout
- Razorpay Payment
- My Orders
- Order Tracking
- Product Reviews

### Admin
- Dashboard
- Product Management
- Category Management
- Customer Order Management
- Update Order Status
- Update Delivery Status

## Project Structure

ShopSphereMVC
- Customer Frontend

ECommerceAPI
- Backend REST API
